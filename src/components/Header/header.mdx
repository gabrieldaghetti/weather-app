# Header

## Descrição
Componente responsável por mostrar as seguintes informações:<br/>
> Cidade <br/> Pais <br/> Data de hoje <br/> Temperatura Atual <br/> Descrição da Condição Climática

## Props
> date: string - Data atual <br/> currentWeather: object - Objeto com os dados climáticos atuais <br/> country: string - País <br/> city: string - Cidade <br/> translateY: number - AnimatedValue usado para as animações de opacidade e transição
