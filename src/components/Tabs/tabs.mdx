# Tabs

## Descrição
Componente responsável por mostrar as seguintes informações:<br/>
> Temperatura Mínima <br/> Temperatura Máxima <br/> Humidade <br/> Velocidade do Vento

## Props
> currentWeather: object - Objeto com os dados climáticos atuais <br/> translateY: number - AnimatedValue usado para as animações de opacidade e transição
