# Weather Forecast

## Descrição
Componente responsável por mostrar a previsão climática semanal, possuindo as seguintes informações:<br/>
> Dias da Semana <br/> Temperatura Média de cada dia <br/> Tipo de Clima representado por ícones

## Props
> dailyWeather: object - Objeto com os dados climáticos semanais
