# Services

## Descrição
Api Open Weather Map juntamente com axios para fazer as requisições.<br/>

## Props
>lat: number - Latitude <br/>
>lon: number - Longitude <br/>

## fetchCurrentWeather
Função que retorna dos dados climáticos em tempo real com base em sua latitude e longitude
> function fetchCurrentWeather(lat: number, lon: number) {}

## fetchDailyWeather
Função que retorna dos dados climáticos dos próximos 6 dias com base na sua latitude e longitude
> function fetchDailyWeather(lat: number, lon: number) {}
